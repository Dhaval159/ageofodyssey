# Assets

## Sprites
All game sprites are stored in `src/assets/sprites/`.

### Player
- `player_idle.png` - Idle animation frames
- `player_run.png` - Running animation frames
- `player_jump.png` - Jump animation frames
- `player_attack.png` - Attack animation frames
- `player_hurt.png` - Hurt/damage animation frames
- `player_death.png` - Death animation frames

### Enemies
- `crab.png` - Small crab enemy
- `bat.png` - Flying bat enemy
- `boar.png` - Wild boar enemy
- `spirit.png` - Forest spirit enemy
- `spider.png` - Cave spider enemy
- `golem.png` - Rock golem enemy

### Bosses
- `cyclops.png` - The Cyclops boss spritesheet

## Tilesets
Stored in `src/assets/tilesets/`.
- `shore.png` - Beach/shore tileset
- `forest.png` - Forest tileset
- `cave.png` - Cave tileset
- `lair.png` - Cyclops lair tileset

## UI
Stored in `src/assets/ui/`.
- `button.png` - Button spritesheet
- `health_bar.png` - Health bar segments
- `inventory_bg.png` - Inventory background
- `icon_sword.png` - Sword icon
- `icon_shield.png` - Shield icon
- `icon_potion.png` - Potion icon
- `icon_key.png` - Key item icon

## Audio
Stored in `src/assets/audio/`.

### Music
- `menu_theme.mp3` - Main menu background music
- `shore_theme.mp3` - Shore level music
- `forest_theme.mp3` - Forest level music
- `cave_theme.mp3` - Cave level music
- `boss_theme.mp3` - Boss battle music

### SFX
- `sword_swing.mp3` - Sword attack sound
- `hit.mp3` - Enemy hit sound
- `player_hurt.mp3` - Player taking damage
- `jump.mp3` - Jump sound effect
- `collect.mp3` - Item collection sound
- `door_open.mp3` - Door/chest opening
- `death.mp3` - Player death sound

## Fonts
Stored in `src/assets/fonts/`.
- `MedievalSharp-Regular.woff2` - Main game font
- `Cinzel-Regular.woff2` - Title/heading font
