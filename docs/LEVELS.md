# Levels

## Overview
Odyssey: The Cyclops is a side-scrolling action-adventure game. Below are the planned levels.

## Level List

### Level 1: The Shore
- Introduction level
- Player washes ashore on the Cyclops' island
- Basic movement and combat tutorial
- Enemies: Small crabs, bats

### Level 2: The Forest
- Dense forest area
- Introduces platforming elements
- Enemies: Wild boars, forest spirits

### Level 3: The Cave
- Underground cave system
- Dark environment, limited visibility
- Puzzles involving light sources
- Enemies: Cave spiders, rock golems

### Level 4: The Cyclops' Lair
- Final boss level
- Confrontation with the Cyclops
- Multi-phase boss fight
